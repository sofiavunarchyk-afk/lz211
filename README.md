Головне завдання програми полягає у зчитуванні набору цілих чисел із зовнішнього текстового файлу та обчисленні добутку всіх від'ємних елементів цього набору. Взаємодія здійснюється через інтуїтивно зрозумілий інтерфейс: після натискання користувачем керуючої кнопки алгоритм автоматично завантажує вміст файлу та повністю виводить його у багаторядкове текстове поле для візуального контролю. Далі програма розбиває зчитаний текст на окремі числові значення, ігноруючи пробіли та переходи на нові рядки, і за допомогою циклу перевіряє кожне число. Усі знайдені від'ємні числа перемножуються між собою, а кінцевий результат цього математичного обчислення виводиться у спеціально відведене текстове поле. Додатково у додатку реалізовано систему обробки виключних ситуацій, яка захищає програму від збоїв і виводить попереджувальні діалогові вікна, якщо потрібний файл відсутній у директорії або якщо його вміст має некоректний формат.
